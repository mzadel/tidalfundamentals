## Installing the tidal haskell libraries

MacOS | Tidal Cycles
https://tidalcycles.org/docs/getting-started/macos_install

(Reconstructed these steps from memory...)
(Can redo the install to confirm)

 - Used stack
     - Stack (Haskell) - Wikipedia https://en.wikipedia.org/wiki/Stack_(Haskell)
     - https://haskellstack.org
     - https://docs.haskellstack.org/en/stable/install_and_upgrade/
 - Can blow away `~/.stack` to reset your install
 - In `~/.stack/global-project/stack.yaml`, used `resolver: lts-18.12`
     - Go to stackage and click on LTS at the top
     - https://www.stackage.org/lts-18.12
 - Installed the global ghc with `stack setup` (I think)
     - Ends up in `~/.stack/programs/x86_64-osx/ghc-8.10.7/bin/ghc`
         - See `stack path`
     - Invoke as `stack ghc` or `stack ghci`
 - Then it seems like I did `stack install tidal`
